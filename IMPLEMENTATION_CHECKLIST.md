# Implementation Checklist

Before merging automation-related changes:

- [ ] Source logged in `docs/automation-change-log.md`
- [ ] Confidence level set correctly (`official`, `github-reference`, `runtime-verified`, `hypothesis`)
- [ ] Any unsupported/suspected flow is marked clearly, not presented as fact
- [ ] `npm run typecheck` passes
- [ ] Relevant changelog entry updated with files changed and summary
- [ ] User-facing report states current confidence level honestly
